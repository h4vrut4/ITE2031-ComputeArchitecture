# ITE2031-ComputerArchitecture

This repository contains lab projects and assignments for **ITE2031 Computer Architecture** at Hanyang University.  
It documents the step-by-step implementation of CPU architectures, progressing from a basic Single-Cycle CPU to an advanced Pipelined CPU equipped with Data Forwarding and Branch Prediction.

---

## 📂 Repository Structure

The repository is organized into four distinct stages based on architectural complexity:

```text
.
├── single_cycle_cpu_ComputerArchitecture/       # Single-Cycle CPU implementation
├── multicycle_cpu_ComputerArchitecture/         # Multi-Cycle CPU implementation
├── pipelined_cpu_ComputerArchitecture/          # Basic 5-stage Pipelined CPU implementation
└── pipelined_cpu_w_forwarding_bp_ComputerArchitecture/ # Final Pipelined CPU with Forwarding & Branch Prediction
